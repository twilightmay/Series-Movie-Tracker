Version 1.6 April 10 2026

New Features

  • Watch Timeline Range Selector: You can now switch the statistics timeline between Lifetime, 1 Year, or 1 Month — making it easier to see your recent watching habits.
  • “Max Time to Finish” & Timer: Set an estimated time (in minutes) for a movie or episode. Long‑press (650ms) on a series card to start a timer. When the time runs out, the tracker automatically advances to the next episode/season or marks the series as completed.
  • Smarter Watch History: Editing the current season, episode, or time while a series is marked as “Watching” now automatically updates the “Last Watched Date” and logs the change in the watch history.
  • Enhanced Timeline Visuals: The watch timeline now features a smooth animated line, gradient area fill, glowing hover effects, and a tooltip that shows detailed per‑day session information. Click any dot to pin the details.
  • Improved Streaming Icon Fallback: If a streaming service icon fails to load, the app falls back to a generic favicon lookup (e.g., google.com/s2/favicons) so you never see broken images.

Improvements

  • Sidebar now uses full width on mobile devices, making it much easier to use on small screens.
  • Touch device optimisations: hover effects are disabled, and all interactive elements have a minimum touch target size of 44px.
  • Images now load with lazy loading and async decoding to prevent blocking the page during initial render.
  • Logo loads eagerly for faster perceived performance.

Bug Fixes

  • Fixed an issue where the timeline would not update correctly when switching between different timeline ranges.
  • Corrected a bug that prevented the “Last Watched Date” from being recorded when editing an existing series.

Coming Soon

  • Full API Integration: Automatic cover, rating, and streaming service detection via OMDB and Watchmode (work in progress).
  • Better Statistics Graphs: More detailed per‑genre and per‑service breakdowns.
  • Custom Filters: Create your own filter rules for advanced organisation.
Version 1.5 March 16 2026

New Features

  • Favourite Actors: You can now add up to 3 favourite actors/voice actors per entry, including their name and a profile image URL — a feature that was long planned and is finally here!
  • Custom Status Badge Colors: In the Custom Theme you can now individually change the color of each status badge — Currently Watching, Want to Watch, and Completed — fully personalizing your tracker
  • Streaming Service Quick-Links: Clicking the streaming service icon on an entry now opens a search directly on that platform so you can jump straight into watching
  • Ratings: You can now rate series and movies on a scale of 1 to 10 (Full stars and half stars), with the average rating displayed prominently
  • Cookie Warning Banner: A cookie/storage usage notice is now shown on first visit and can be dismissed permanently
  • Auto-Save While Editing: Form fields now auto-save changes after 1 second of no typing when editing existing entries — a green ✓ confirms the save, and you no longer need to press Save for every small change
  • Auto-Update Last Watched Date: When you update the current season or episode of a series you are watching, the Last Watched Date is automatically set to today
  • Auto-Expand Totals: If you set the current season/episode higher than the stored total, the total is automatically updated to match

Improvements

  • Full Mobile Responsiveness Overhaul: The tracker now has thorough responsive breakpoints for large tablets (1400px), tablets (1023px), mobile (767px), small mobile (480px), and landscape orientation — everything from the header to modals and the sidebar adapts cleanly
  • Touch Device Optimizations: Hover effects are disabled on touch screens to avoid sticky hover states, and all buttons now have a minimum touch target size of 44px
  • Page Load Optimization: The logo now loads eagerly for faster perceived load times, and images across the app use lazy loading with async decoding to prevent blocking the page
  • Sidebar on Mobile: The sidebar now takes the full width on small screens instead of a fixed 350px, making it much easier to use on phones
  • Font Smoothing: Added webkit and moz font smoothing for crisper text rendering on all devices
  • Horizontal Scroll Prevention: The body no longer produces a horizontal scrollbar on narrow screens

Bug Fixes

  • Profile pictures now fall back to a placeholder image if the URL fails to load instead of showing a broken image icon
  • Closing the series modal by clicking outside now properly resets the rating display back to 0
  • Import options modal no longer accidentally triggers the content-type selection dialog

Coming Soon

  • Better Statistics: Enhanced statistics with graphs and more detailed per-genre and per-streaming-service breakdowns
  • API Integration (Full): Completing OMDB and Watchmode API support for fully automatic cover, rating and streaming service detection
  • Better Color Picker for Themes: Improved design of the custom color picker UI

Ideas Under Consideration

  • More Content Orders / Grid Layouts - Different grid arrangements beyond the current list view

Version 1.0 Initial Release

Hello Fellas! Today I want to show you my newest project - Serien.html. Here you can create your own list of series you are watching, have watched, or want to watch.

Core Features

  • Series Tracker Only: Track TV series with series-specific fields like seasons, episodes, and progress tracking
  • Multiple Status Categories:
    • Currently Watching (red badge)
    • Want to Watch (blue badge)
    • Completed (green badge)
  • Comprehensive Series Information: Title, cover image, genre, language, IMDB rating, season/episode progress, total seasons/episodes, and time spent watching
  • Interactive UI: Expandable cards with "More" button for extra details, edit and delete options per series
  • Three View Sizes: Small, Normal, and Large display options
  • Priority Watchlist (Drag & Drop): Add series to a queue, reorder by dragging, separate "Watchlist" section

Search & Filtering

  • Live search across all titles and genres
  • Filter by status (All, Watching, Want, Completed)
  • Sort options: Default order, Title (A-Z / Z-A), Most/Least time spent

Data Management

  • Add new series via modal form
  • Edit existing series
  • Delete series with confirmation
  • All data persists in browser Cookies
  • No login required - works offline

Progress Tracking

  • Visual progress indicators (season/episode)
  • Time spent displayed in hours/minutes
  • Progress badges and status labels
  • Timeline display (S1 E5 format)

Use Cases

  • Track what TV series you're currently watching
  • Maintain a watchlist of series you want to watch
  • Record completed series
  • Organize your viewing queue with drag & drop
  • Quick access to series details and progress

Interface Design

  • Single dark theme only (no theme switching in v1.0)
  • Clean, focused design for series tracking
  • Responsive layout that works on mobile and desktop
  • No streaming service icons or selection (added in v1.1)

Technical Features

  • Pure HTML/CSS/JavaScript implementation
  • No external dependencies
  • Browser-based storage
  • Drag & drop functionality
  • Form validation
  • Responsive design
